# Instalação — react-icons (OBRIGATÓRIO)

Execute este comando na pasta `client` antes de rodar o projeto:

```bash
cd client
npm install react-icons
```

Ou na raiz do projeto:

```bash
npm install --prefix client react-icons
```

Depois de instalar, inicie normalmente:

```bash
npm run dev
```

## Por que?
Os arquivos `Layout.js`, `ChamadaPage.js`, `AuditoriaPage.js` e `AdminRelatoriosPage.js`
agora usam `react-icons/fi` e `react-icons/gi` para ícones.
